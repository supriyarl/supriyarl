## Professional Summary
Electronics and Communication Engineer specializing in **VLSI design, RTL development, and neuromorphic hardware systems**, with **hands-on R&D experience at ULOG3**. Actively contributed to the **design and development of a single-core neuromorphic chip**, implementing **LIF neuron models, synaptic traversal architectures, and event-driven processing pipelines**. Experienced in translating algorithmic neuron behavior into **cycle-accurate digital hardware** using Verilog, with strong interest in **low-power, brain-inspired, and research-driven computing architectures**.
